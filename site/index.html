Pick your flavor

18 combinations of desktop × browser × flatpak, built from one matrix.

1. Switch a running bootc system

Atomic upgrade in-place. Reboot to apply.

bootc switch ghcr.io/vaskozl/niri-firefox-flatpak:latest

2. Try it in a container

Pull the image and poke around — no commitment.

podman run --rm -it ghcr.io/vaskozl/niri-firefox-flatpak:latest /bin/bash
3. Install to disk

Boot any Linux live USB. Pull the flavor image, then run bootc install to-disk against the target device.

podman run --rm --privileged --pid=host -v /dev:/dev -v /:/target \
  ghcr.io/vaskozl/niri-firefox-flatpak:latest \
  bootc install to-disk /dev/sdX

Replace /dev/sdX with your target disk. This wipes the disk.

Why Sheltie Linux

Five things you get out of the box. The rest of the page is evidence for them.

Zero CVE, minimal

Built from Wolfi. The comparison table below is the receipt.

Atomic & auto-updating

bootc updates the base image; podman auto-updates containers. No package drift.

No installer

Copy the bootc image straight to disk with bootc install to-disk once booted. No anaconda, no Calamares.

Immutable root

System files are read-only. User state lives in /var and $HOME.

18 flavors from one matrix

Pick desktop × browser × flatpak. Every combination is built and signed.

OCI all the way down

Same image you boot is the one you can podman run. Inspect it like any other container.

Frequently asked

Why "Sheltie Linux"?

Inspired by Puppy Linux flavors. Same spirit (small, friendly, directly usable, atomic), different breed. Our images are bootc OCI containers — pull and run, no installer needed.

How do I update?

You don't. bootc updates the base image; podman auto-updates containers. Reboot to pick up new bootc layers.

Can I customise?

Build your own flavor: fork the repo, add a flavors/<name>.yaml, open an MR.

Where do I report bugs?

File an issue at gitlab.sko.ai/doudous/apkontainers/-/issues.

Is this an image factory?

No. Sheltie Linux ships a fixed matrix of 18 flavors. The picker exposes that matrix; it doesn't generate custom images.
